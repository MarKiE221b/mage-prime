# Hero Video Setup

Place your construction background video in this folder and name it:

```
hero-video.mp4
```

The path referenced in HeroSection.jsx is `/hero-video.mp4`, which maps to this `public/` directory.

Recommended specs:
- Format: MP4 (H.264)
- Resolution: 1920×1080 or higher
- Duration: 15–60 seconds looping
- Keep file size under 20 MB for fast initial load
